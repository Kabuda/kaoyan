# 11408 二战私人学习驾驶舱设计文档

## 背景

本项目是一个面向单人使用、可公网部署的考研学习网站。用户是二战考生，考试科目为 11408，需要一个每天可用的学习驾驶舱，用来完成每日规划、学习计时、记录沉淀和阶段复盘。

第一版目标不是做完整题库或 AI 备考平台，而是先做一个能稳定服务日常学习的系统：打开网站后能知道今天学什么，计时后能留下准确记录，复盘时能看到英语、数学、408 的投入是否符合二战提分策略。

## 用户目标与基线

一战成绩：

- 政治：64
- 英语：46
- 数学：74
- 408：83

二战目标总分：330。

第一版默认目标分：

- 政治：65
- 英语：55
- 数学：110
- 408：100

默认备考策略：

- 数学是第一主攻科目，从 74 拉到 110，需要高频任务、长时段计时和复盘。
- 英语是风险科目，从 46 拉到 55，要求每日不断档。
- 408 从 83 拉到 100，按数据结构、组成原理、操作系统、计算机网络拆分记录。
- 政治从 64 稳到 65，第一版以保持节奏为主。

这些目标分数必须支持在设置页修改。

## 用户体验原则

- 首页优先回答三个问题：今天学什么，今天学了多久，弱项有没有照顾到。
- 所有任务都必须能快速创建、开始计时、结束记录。
- 统计不追求花哨，优先帮助二战考生发现计划偏差。
- 公网部署必须默认有登录保护，不能裸奔。
- 第一版允许手动录入，不依赖题库、爬虫或第三方服务。

## 功能设计

### 登录保护

系统只支持一个管理员账号。

要求：

- 使用用户名和密码登录。
- 密码使用哈希存储。
- 登录成功后返回 JWT。
- 前端在请求受保护接口时带上 token。
- 禁止开放注册。
- 除健康检查和登录接口外，后端 API 默认需要鉴权。
- 初始管理员账号通过环境变量配置。

### 备考档案

备考档案保存二战目标和一战基线。

字段：

- 一战政治分数
- 一战英语分数
- 一战数学分数
- 一战 408 分数
- 目标政治分数
- 目标英语分数
- 目标数学分数
- 目标 408 分数
- 目标总分
- 考试日期
- 每日目标学习分钟数
- 当前弱项说明

默认值：

- 一战政治 64
- 一战英语 46
- 一战数学 74
- 一战 408 83
- 目标政治 65
- 目标英语 55
- 目标数学 110
- 目标 408 100
- 目标总分 330

### 首页仪表盘

首页显示当天最重要的信息。

模块：

- 今日学习总时长
- 今日目标完成率
- 今日任务列表
- 当前运行中的计时
- 英语今日是否安排
- 数学今日是否安排
- 408 今日是否安排
- 本周各科投入占比
- 最近 7 天学习趋势
- 连续学习天数

提醒规则：

- 如果当天没有英语任务，显示英语不断档提醒。
- 如果本周数学学习时长低于总时长的 30%，显示数学主攻提醒。
- 如果本周 408 学习时长低于总时长的 20%，显示 408 投入提醒。
- 如果今日学习时长低于每日目标的 50%，显示进度提醒。

### 每日规划

用户可以按日期管理学习任务。

任务字段：

- 日期
- 科目
- 模块
- 标题
- 任务类型
- 预计分钟数
- 实际分钟数
- 优先级
- 状态
- 备注

科目：

- 政治
- 英语
- 数学
- 408

模块：

- 政治：选择题、背诵、真题、模拟、复盘
- 英语：单词、阅读、翻译、作文、完形、新题型、复盘
- 数学：高数、线代、概率论、真题、错题、复盘
- 408：数据结构、组成原理、操作系统、计算机网络、真题、复盘

任务类型：

- 补弱
- 刷题
- 背诵
- 听课
- 总结
- 复盘

任务状态：

- 未开始
- 进行中
- 已完成
- 已跳过
- 已延期

操作：

- 新增任务
- 编辑任务
- 删除任务
- 完成任务
- 跳过任务
- 延期到指定日期
- 从任务开始计时
- 一键生成今日基础计划

今日基础计划默认生成：

- 英语 60 分钟，类型为补弱或刷题
- 数学 180 分钟，类型为补弱或刷题
- 408 120 分钟，类型为刷题或复盘
- 政治 45 分钟，类型为背诵或选择题

默认计划可以在设置中调整。

### 学习计时

学习计时支持围绕任务进行。

要求：

- 一个时间点只能有一个运行中的计时。
- 计时必须关联任务，也允许创建临时任务后计时。
- 支持开始、暂停、继续、结束。
- 结束时生成学习记录。
- 结束时可填写学习总结、卡点、完成质量。
- 如果前端关闭页面，后端仍保留运行中计时状态。

完成质量：

- 低
- 中
- 高

计时模式：

- 正计时
- 番茄钟

第一版番茄钟只做本地倒计时体验，最终记录仍以实际开始和结束时间为准。

筛选：

- 日期范围
- 科目
- 模块

- 完成质量

操作：

- 手动补录记录

### 统计复盘

- 最近 7 天趋势
- 最近 30 天趋势
- 英语连续学习天数
- 数学周投入占比
- 408 周投入占比

图表：

- 各科投入饼图或环图
- 预计 vs 实际柱状图
- 408 子模块投入柱状图

## 错误处理

后端：

- 未登录返回 401。
- 无权访问资源返回 403。
- 资源不存在返回 404。
- 请求字段错误返回 422。
- 运行中计时冲突返回 409。
- 删除或更新不属于当前用户的数据返回 404，避免泄露资源存在性。

前端：

- 401 自动清理 token 并跳转登录。
- 表单错误在字段附近显示。
- 计时冲突显示明确提示。
- 网络错误显示重试提示。

## 安全与部署

安全要求：

- 密码必须哈希存储。
- JWT_SECRET 必须来自环境变量。
- 生产环境 CORS 必须配置为具体域名。
- 后端不得在生产环境开启 debug。
- MySQL 密码来自环境变量。
- `.env` 不提交。

部署文件：

- `docker-compose.yml`
- `backend/Dockerfile`
- `frontend/Dockerfile`
- `.env.example`

第一版 Docker Compose 服务：

- `mysql`
- `backend`
- `frontend`

后续公网域名接入时，可以在服务器上增加 Nginx 或 Caddy 处理 HTTPS。
